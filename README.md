# 🤖 Telegram Bot Business - Git Sync Project

  Bot Business অ্যাপের Git Sync ফিচারের জন্য তৈরি।

  ## 📁 ফাইল স্ট্রাকচার
  ```
  ├── bot.json       → বটের কনফিগারেশন
  ├── flows.json     → বটের ফ্লো ও বাটন
  ├── messages.json  → বটের বার্তা
  ├── config.json    → অ্যাডমিন ও পেমেন্ট সেটিং
  ```

  ## 🔄 Import নিয়ম (Bot Business)

  **Step 1:** এই repo-র Settings → Deploy keys → Add deploy key
  Bot Business অ্যাপের SSH key paste করুন + Allow write access

  **Step 2:** Bot Business → Tools → Git sync
  Git repository তে দিন:
  ```
  git@github.com:riad00956/bot-business-project.git
  ```
  তারপর **"Import from git"** চাপুন ✅
